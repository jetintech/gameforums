
# GameForums - Front-End Website

## Description
GameForums is a complete front-end gaming website template built with HTML, CSS, and JavaScript. Designed for gaming communities or starter download sites.

## Files Included:
- index.html
- style.css
- script.js
- README.txt

## How to Use:
1. Unzip the GameForums.zip file.
2. Open `index.html` in your browser to view the site.
3. You can edit the content in `index.html`, styles in `style.css`, and effects in `script.js`.

## Notes:
- All images are royalty-free placeholders from Unsplash.
- No backend is included (static only).
